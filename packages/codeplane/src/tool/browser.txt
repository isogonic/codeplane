Browser automation tool providing full Chrome control via DevTools Protocol.

Use this tool to:
- Navigate any URL
- Take screenshots for visual inspection
- Get a snapshot of interactive elements with refs (@e1, @e2) for clicking/typing
- Click elements by stable snapshot refs, CSS selector, or viewport coordinates
- Hover elements and drag between viewport coordinates
- Type text into input fields, contenteditable elements, or focused coordinates
- Press keys and shortcuts such as Enter, Escape, Tab, Cmd+L, Ctrl+R, and ArrowDown
- Scroll and wait with a verification screenshot after each action
- Go back, go forward, reload, and close the isolated browser
- Execute JavaScript in the page
- Get console logs and network requests
- Extract page HTML for analysis

The browser session persists across calls - same tab, same cookies, same state.
Run snapshot after major DOM changes to refresh refs, then use refs for safer actions than raw coordinates.

On desktop, Chrome/Chromium is auto-detected and launched on a private DevTools port if needed.
The browser uses virtual DevTools mouse and keyboard events, so it does not move the user's real desktop cursor.
