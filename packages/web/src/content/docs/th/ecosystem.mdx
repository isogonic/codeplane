ชุดโครงการชุมชนที่สร้างขึ้นบน CodePlane

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ CodePlane ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-codeplane](https://github.com/awesome-codeplane/awesome-codeplane) และ [codeplane.cafe](https://codeplane.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
