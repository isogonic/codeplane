Documentation

Everything Codeplane.

From your first install to writing your own plugin. Every CLI flag, every config knob, every surface (terminal, desktop, web, mobile) — kept in sync with the code.

Get started

  • InstallGet the binary or desktop app on macOS, Linux, Windows. iOS via TestFlight; Android coming soon.
  • Quick startFrom `codeplane web` to your first agent reply, in under a minute.
  • ConfigurationThe codeplane.jsonc reference: providers, agents, permissions, MCP, server, runtime tuning.
  • ProvidersModels, auth methods, OAuth flows, API-key setup, custom OpenAI-compatible endpoints.

Surfaces

  • TUIFull-screen terminal interface in any shell.
  • DesktopNative macOS, Linux, and Windows app. Auto-updates via electron-updater.
  • WebSame UI in any browser. Nothing to install client-side.
  • MobileNative iOS shell on TestFlight. Android in development.
