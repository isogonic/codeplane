Codeplane Docs

This static entry point exists so Codeplane links can resolve on codeplane.cc while the full docs site evolves.

Install

curl -fsSL https://codeplane.cc/install | bash

Configuration

Project and global config files can reference the hosted schema endpoint.

{
  "$schema": "https://codeplane.cc/config.json",
  "commit": {
    "coauthor": true
  }
}

Commit co-authoring

Enable the Codeplane co-author setting to append this trailer to commits made through Codeplane's native git tool. 

Co-Authored-By: Codeplane <noreply@codeplane.cc>
