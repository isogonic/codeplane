Keybinds

Every shortcut across the web, desktop, and TUI front-ends. On macOS use ⌘ where the table says Mod; on Windows / Linux use Ctrl.

Global (every surface)

KeyAction
Mod+KOpen the quick switcher (sessions, projects, files, commands).
Mod+NNew session in the current project.
Mod+BToggle the left sidebar.
Mod+\Toggle the right side panel (review / timeline / terminal).
Mod+/Search the active session.
Mod+,Open settings.
Mod+Shift+PCommand palette (every command, every keybind).
?Show the keybind overlay (web + TUI only).

Composer

KeyAction
EnterSend message.
Shift+EnterNewline.
Mod+EnterSend (when the message is multi-line and Enter would newline).
EscCancel a streaming reply.
↑ / ↓Cycle through your previous messages.
/Open the slash-command popover.
@Open the file / agent mention popover.

Timeline

KeyAction
j / kScroll down / up by one message (TUI).
Mod+RRevert to the previous user message.
Mod+Shift+RRetry the current turn with the same prompt.
Mod+LClear the visible buffer (history is kept).
Mod+CCopy the focused message.

Customizing

Override any binding in ~/.codeplane/keybinds.json:

{
  "command.session.new":      ["mod+t"],
  "command.sidebar.toggle":   ["mod+\\"],
  "composer.send":            ["mod+enter"]
}

The full list of command IDs is in the in-app command palette (Mod+Shift+P) — each entry shows its ID alongside its current binding.
