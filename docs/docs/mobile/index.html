Mobile

iOS and Android shells that wrap the Codeplane web UI. The agent runs on a Codeplane server you already host — the phone is a thin client.

Install

iOS — TestFlight beta, or sideload the .ipa from releases (look for v<x.y.z>-mobile).

Android — Google Play, or adb install the APK from the same release.

First launch

The shell asks for a Codeplane server URL. Three ways to point it at one:

  1. QR code. On your laptop, run codeplane serve --share. The CLI prints a QR — scan it from the mobile app's Add server screen.
  2. URL paste. If you self-host (see Self-hosting), paste the public URL.
  3. Local network. The shell auto-discovers Codeplane servers on the same Wi-Fi via mDNS — they show up in the picker without typing.

What works on mobile

  • Reading + scrolling sessions, all message types (text, diffs, tool calls, terminal output).
  • Sending follow-ups + voice-to-text via the system keyboard.
  • Re-ordering the queued follow-up list by drag.
  • Reviewing diffs (single-column on phone, side-by-side on tablet).
  • Live activity on iOS — a long-running session shows progress on the lock screen + Dynamic Island.

What's deliberately reduced

Phone screens aren't great for some operations, so the mobile shell hides them:

  • The terminal pane is read-only (you can't type into a running shell session).
  • File-tree + multi-file picker is collapsed to a search field.
  • Plugins that draw custom UI panels render in a simplified form.

Live activity (iOS)

When a session is busy, the iOS shell publishes a Live Activity to the lock screen + Dynamic Island so you can monitor progress without unlocking. Toggle it in Settings → Sessions → Live activity. Live Activities respect Focus modes the same way as any other app.

Offline behaviour

Sessions live on the server. When the phone is offline, the mobile shell shows a cached copy of the most recent timeline and queues outgoing messages — they send the moment connectivity returns. Auto-reconnect uses exponential backoff (1s, 2s, 5s, 15s, then every 30s).
