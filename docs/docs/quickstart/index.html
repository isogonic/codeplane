Quick start

From a fresh install to your first agent reply. Sixty seconds, no editor lock-in, no signup.

1. Install the CLI

One line on macOS or Linux:

curl -fsSL https://codeplane.cc/install | bash

On Windows in PowerShell:

irm https://codeplane.cc/install | iex

Full per-platform options — Homebrew, npm, native installers, mobile — at Install.

2. Provide an API key

Codeplane needs at least one model provider configured. The fastest path is an environment variable:

export ANTHROPIC_API_KEY=sk-ant-...
# or
export OPENAI_API_KEY=sk-...
# or
export OPENROUTER_API_KEY=sk-or-...

That's enough for a first session. To pin providers and models permanently, see Configuration.

Local model? Point Codeplane at any OpenAI-compatible endpoint (Ollama, vLLM, LM Studio, llama.cpp) by setting OPENAI_BASE_URL. The shipped codeplane.json picks up the env var automatically.

3. Launch a server

Pick the surface that matches what you're doing.

Web (recommended first time)

codeplane web

Boots a local server, opens your default browser. The first launch creates a workspace at the directory you ran the command from.

Terminal (no GUI)

codeplane tui

Full-screen text UI. Same agent, same sessions — useful over SSH, on a low-power box, or when you'd rather stay in the terminal.

Headless (server-only)

codeplane serve --port 4096

No browser, no TUI — just the HTTP server. Point a desktop / mobile / web client at http://localhost:4096 to connect. Useful for VPS setups; see Self-hosting.

4. Send your first message

Type into the composer:

What's the structure of this repo? Give me a 3-line summary.

Watch the agent stream a reply. It will read files, run a few commands, and answer. The first time it edits a file or runs a shell command, you'll get a permission prompt — that's the per-directory approval layer. See Permissions for how to relax it.

5. Pick up the same session anywhere

The session ID lives on the server. Once you've started a thread in the web UI, you can reopen it in:

  • The desktop app — point it at http://localhost:4096 via Add Server
  • The TUI — codeplane tui picks up the active server automatically
  • The mobile app — scan the QR code shown by codeplane serve --share from your phone
