Install

Get Codeplane on your machine.

Codeplane ships as a single standalone binary (no Node or runtime required) plus native shells for desktop and mobile. Pick the path that fits your platform.

macOS

One-line installer for the CLI + standalone .dmg for the desktop app. Both are signed for Apple Silicon and Intel.

CLI · scripted installer

curl -fsSL https://codeplane.cc/install | bash

Detects darwin-arm64 or darwin-x64, downloads the matching ~94 MB binary into ~/.codeplane/bin/, symlinks codeplane into /usr/local/bin (or $PATH equivalent). Pin a version with -v 28.1.20; skip the PATH edit with --no-modify-path.

CLI · Homebrew

brew install devinoldenburg/codeplane/codeplane

Same binary, managed via Homebrew so brew upgrade picks up new releases. Tap lives at github.com/devinoldenburg/homebrew-codeplane.

Desktop · .dmg

Download the right disk image for your chip and drag Codeplane.app into /Applications:

  • Apple Silicon (M1 / M2 / M3 / M4)
  • Intel

The desktop app updates itself silently through electron-updater after first launch — you don't need to come back here for new versions.

Gatekeeper note. If you sideload the .dmg from a CI build (not signed with our Developer ID), strip the quarantine flag once: xattr -dr com.apple.quarantine /Applications/Codeplane.app.

Linux

Standalone binary plus desktop packages for the major distributions. Tested on x86_64; arm64 binaries ship alongside.

CLI · scripted installer

curl -fsSL https://codeplane.cc/install | bash

Detects linux-x64 or linux-arm64, drops the binary into ~/.codeplane/bin/, edits your shell rc to put it on PATH. Add -v <version> to pin, --no-modify-path to skip the PATH edit.

Desktop · AppImage / deb / tar.gz

  • .AppImage — universal, no install needed (chmod +x and run).
  • .deb for Debian / Ubuntu / Mint — sudo dpkg -i codeplane-desktop-*-linux-amd64.deb
  • .tar.gz for any other distro — extract anywhere, run ./codeplane-desktop.

Browse all Linux artefacts at /releases/latest.

Headless server

codeplane serve --port 4096 --hostname 0.0.0.0

Run on any Linux box to expose Codeplane on your LAN or behind a reverse proxy. Use systemd to keep it alive across reboots — see Self-hosting.

Windows

NSIS installer plus a portable .zip. WSL2 users can also follow the Linux instructions for a CLI install.

Desktop · installer

Download and run:

  • codeplane-desktop-windows-x64.exe — NSIS installer (signed when the release ships a code-signing cert).

Installs into %LOCALAPPDATA%\Programs\Codeplane and registers a Start menu entry. Updates land silently via electron-updater.

CLI · PowerShell

irm https://codeplane.cc/install | iex

PowerShell-friendly installer (mirrors the curl flow on macOS / Linux). Drops the binary in %USERPROFILE%\.codeplane\bin\codeplane.exe and adds it to PATH for the current user.

CLI · WSL2

curl -fsSL https://codeplane.cc/install | bash

Inside a WSL2 shell, the regular Linux installer works end-to-end. The CLI runs from the WSL filesystem; the desktop / web UI from Windows side talks to it via localhost.

npm / Bun

The same single-file native binary, published under codeplane on npm — handy when your tooling lives in ~/.npm-global or you want bunx on-demand runs.

Global install

npm install -g codeplane
# or
bun install -g codeplane
# or
pnpm add -g codeplane

One-shot run (no install)

bunx codeplane web
# or
npx codeplane@latest web

Useful for trying it without polluting your global package list. The first run downloads the platform-specific binary; subsequent runs reuse the cached one.

SDK + plugin packages

  • @codeplane-ai/sdk — TypeScript client for the Codeplane HTTP API.
  • @codeplane-ai/plugin — base classes for writing Codeplane plugins.

Both ship the same version number as the CLI; see SDK docs.

Mobile

Native iOS + Android shells that wrap the Codeplane web UI. Pair them with a Codeplane server you already run on your laptop / VPS — the shell handles sign-in, picker, live activities, and offline cache.

iOS — TestFlight

Join the TestFlight beta

Public beta channel. Codeplane lands in the App Store at apps.apple.com/app/codeplane once it's out of beta.

iOS — sideload .ipa

Each mobile release ships an .ipa + .xcarchive.zip at /releases (look for the v<x.y.z>-mobile tag). Pair with AltStore / Sideloadly / a free Apple ID and you can install it without a paid dev account.

Android — Play Store

Open Google Play

Android — APK / AAB sideload

adb install Codeplane-Android-<version>.apk

APK + AAB for every release are at /releases under the v<x.y.z>-mobile tag.

Verifying the install

Same check on every platform — print the version, run a smoke command.

codeplane --version
codeplane --help
codeplane web --port 4096

The first command should print the semantic version (28.2.2 at the time of writing). The second lists every subcommand. The third boots a local server and opens the web UI in your default browser.

Uninstalling

No registry rot, no system services. Codeplane is a single binary + a config dir.

macOS / Linux

rm -rf ~/.codeplane                              # binary + cache
rm -rf ~/Library/Application\ Support/Codeplane  # macOS user data
rm -rf ~/.config/codeplane                       # Linux user data
sudo rm /usr/local/bin/codeplane                 # PATH symlink

Windows

Uninstall the desktop app via Settings → Apps. Remove the CLI binary at %USERPROFILE%\.codeplane\ and the user data at %APPDATA%\Codeplane\.

Mobile

Long-press the app icon → Delete. All session state lives on the Codeplane server you connected to; the shell only caches preferences locally.
