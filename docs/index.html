A coding agent that lives
everywhere you code.

Codeplane is one server, four front-ends — terminal, desktop, web, mobile. Plug in your model, point it at your repo, and pick up the same session from any device. Self-hosted, open, fast.

Install Codeplane Read the docs
$ curl -fsSL https://codeplane.cc/install | bash
# Detecting platform: darwin-arm64
# Downloading codeplane v28.2.2 (94 MB)
# Installing to ~/.codeplane/bin
# Symlinking → /usr/local/bin/codeplane
 
$ codeplane web
# Started server on http://localhost:4096
# Open browser at http://localhost:4096

Why Codeplane

Built for the way you actually work.

Any model

OpenAI, Anthropic, OpenRouter, Ollama, vLLM, custom OpenAI-compatible endpoints — switch providers per session.

Real reasoning

Streamed thinking summaries, queued follow-ups you can reorder by drag, per-session memory and rules.

Tool-rich

File edits, shell, search, git, ghostty-driven terminals, browser previews — all native, no MCP needed.

MCP-ready

Plug in any Model Context Protocol server. Bundled with Filesystem, GitHub, Sequential Thinking out of the box.

Yours alone

Run locally or self-host. Codeplane never reports telemetry; your sessions never leave your hardware.

Open source

MIT-licensed. Read the code, file an issue, ship a PR — every line is at github.com/devinoldenburg/codeplane.

Sixty seconds to running

Install once. Use everywhere.

macOS & Linux

curl -fsSL https://codeplane.cc/install | bash

Detects your platform automatically, installs into ~/.codeplane/bin, and symlinks the codeplane binary into your PATH.

npm / Bun

npm install -g codeplane
# or
bun install -g codeplane

Same binary, packaged for the Node ecosystem. Useful if you already manage tooling through ~/.npm-global.
