# Tetris

Simple HTML/CSS/JS Tetris implementation in a timestamped temp folder.

## How to run

- Open `index.html` in your browser.

## Controls

- `Arrow Left` / `Arrow Right`: move piece
- `Arrow Down`: soft drop
- `Arrow Up`: rotate
- `Space`: hard drop
- `P`: pause/resume
- `R` or **Restart** button: restart game
