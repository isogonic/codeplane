# Classic Tetris

This folder contains a self contained Tetris game built with plain HTML, CSS, and JavaScript.

Run it by opening `index.html` in a browser.

Controls:

- Left/Right arrows: move piece
- Up arrow: rotate
- Down arrow: soft drop
- Space bar: hard drop
- P: pause/resume
- R: restart

Folder uses the timestamp format used by the repository (`YYYYMMDD-HHMMSS`).
